# Task 2 - Temperature Converter

This is a responsive web app that converts temperature values between Celsius, Fahrenheit, and Kelvin.

## Features
- Convert from any of the 3 units
- Clean and responsive UI
- JavaScript-based logic

## Files Included
- `index.html` – Layout
- `style.css` – Styling
- `script.js` – Functionality

## Tools Used
- HTML
- CSS
- JavaScript
